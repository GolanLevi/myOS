# User-specific instruction template

Create a file in this directory, for example:
- `amit.md`
- `golan.md`
- `ops.md`

Use it only for non-secret preferences, role emphasis, and working style.

Suggested fields:
- role / ownership
- preferred review style
- main focus areas
- what to escalate quickly
- communication expectations
- anything that helps a coding assistant support this person better

Do not put secrets here.
